```  
def test():  
    for i in range(100000):  
        i*5  
  
%timeit test()  
2.97 ms ± 26.7 µs per loop (mean ± std. dev. of 7 runs, 100 loops each)  
  
  
%%timeit  
  
import pandas as pd  
  
df = pd.DataFrame({'a': range(100000)})  
df['b'] = df['a'] * 5  
  
657 µs ± 6.73 µs per loop (mean ± std. dev. of 7 runs, 1000 loops each)  
```
